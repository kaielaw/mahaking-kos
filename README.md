# Mahaking Kos - Project Sistem Database I Praktikum
## Deskripsi
Mahaking Kos merupakan website pencarian kos yang dirancang untuk membantu pengguna dalam menemukan tempat kos secara online dengan mudah dan efisien. Website ini menyediakan berbagai informasi terkait kos, seperti detail lokasi, harga kamar, fasilitas, foto kos, serta rating dan review dari pengguna lain.
Website ini memiliki dua jenis pengguna, yaitu pencari kos dan pemilik kos. Pencari kos dapat mencari kos, melihat detail kos, memberikan review, serta menyimpan kos favorit ke wishlist. Sementara itu, pemilik kos dapat mengelola data kos, kamar, fasilitas, serta informasi properti yang dimiliki.
Mahaking Kos juga mendukung pengelolaan data booking dan pembayaran untuk pengembangan fitur pemesanan kamar kos secara online. Namun, fitur booking bersifat situasional dan akan diimplementasikan apabila terdapat waktu yang cukup.
Berikut ini fitur utama dalam website Mahaking Kos.
- Pencarian kos berdasarkan lokasi.
- Informasi detail mengenai kos dan kamar.
- Pengelolaan data kos dan kamar.
- Wishlist kos favorit.
- Review dan rating kos.
Dengan adanya website ini, proses pencarian kos menjadi lebih modern, terorganisir, dan mudah diakses kapan saja melalui website.
## Latar Belakang
Perkembangan teknologi informasi yang semakin pesat telah membawa banyak perubahan dalam berbagai aspek kehidupan, termasuk dalam bidang pencarian dan pengelolaan tempat tinggal sementara seperti kos. Kebutuhan terhadap tempat tinggal sementara terus meningkat, terutama di daerah Jatinangor dan wilayah sekitar kampus. Mahasiswa, pekerja, maupun pendatang dari luar daerah sering kali mengalami kesulitan dalam mencari kos yang sesuai dengan kebutuhan dan anggaran mereka.
Saat ini, proses pencarian kos masih banyak dilakukan secara manual melalui media sosial, grup percakapan, brosur, atau survei langsung ke lokasi. Cara tersebut sering kali memerlukan waktu yang cukup lama dan kurang efisien karena informasi yang diperoleh tidak selalu lengkap dan terstruktur. Pencari kos sering mengalami kendala dalam memperoleh informasi mengenai harga kamar, fasilitas yang tersedia, lokasi, jumlah kamar kosong, maupun aturan kos yang berlaku.
Saat ini, promosi kos umumnya masih dilakukan secara terpisah melalui berbagai media sosial, seperti Instagram, WhatsApp, Facebook, ataupun grup percakapan, sehingga informasi kos tidak terpusat dalam satu platform. Hal tersebut menyebabkan pencari kos harus mencari informasi dari berbagai sumber yang berbeda.
Berdasarkan permasalahan tersebut, diperlukan sebuah sistem pendataan dan pencarian kos berbasis web yang dapat membantu proses pengelolaan dan penyampaian informasi secara lebih efektif dan terorganisir. Sistem ini dirancang untuk menyediakan informasi kos secara lengkap, mempermudah pencari kos dalam menemukan tempat tinggal yang sesuai, serta membantu pemilik kos dalam mengelola data kos dan kamar dengan lebih efisien.
Selain itu, sistem ini juga menjadi implementasi penerapan konsep sistem database, khususnya dalam perancangan entity relationship diagram (ERD), relasi antar tabel dan pengelolaan data berbasis basis data relasional. Dengan adanya sistem ini, diharapkan proses pencarian dan pengelolaan kos dapat dilakukan dengan lebih mudah, cepat, dan terstruktur.
## Tujuan dan Manfaat

Berikut ini tujuan dari website Mahaking Kos.
- Membantu pencari kos menemukan kos sesuai kebutuhan.
- Membantu pemilik kos mempromosikan kost.

#### Pencari Kos
- Mempermudah pencarian kos.
- Mendapatkan informasi kos secara lengkap.
- Dapat melakukan survey secara online.
- Dapat membuat daftar wishlist.
#### Pemilik Kos
- Mempermudah pengelolaan data kos.
- Membantu memantau booking dan pembayaran.
- Mempermudah promosi kos.

- Menjadi implementasi pembelajaran sistem database.
- Melatih perancangan ERD dan relational database.

## Bahasa Pemrograman, Framework, dan Software yang Digunakan
### Bahasa Pemrograman
Dalam pengembangan website Mahaking Kos, kami menggunakan beberapa bahasa pemrograman untuk membangun sistem berbasis web. Berikut bahasa pemrograman yang digunakan.
#### PHP
PHP digunakan sebagai bahasa pemrograman utama dalam pengembangan backend website.
#### HTML
HTML digunakan untuk membangun struktur halaman website, seperti tampilan halaman utama, detail kos, login, wishlist, review, dan dashboard.
#### CSS
CSS digunakan untuk mengatur tampilan dan desain antarmuka website agar lebih menarik dan rapi.
### Framework dan Software
Berikut framework dan software yang digunakan dalam proses perancangan dan pengembangan website Mahaking Kos.
#### Laravel
Laravel digunakan sebagai framework utama dalam pengembangan website Mahaking Kos.
#### Draw.io
Draw.io digunakan untuk membuat entity relationship diagram (ERD) serta relasi antartabel.
#### MySQL
MySQL digunakan sebagai Database Management System (DBMS) untuk menyimpan seluruh data pada website Mahaking Kos, seperti data user, kos, kamar, fasilitas, dan review.
#### Visual Studio Code
Visual Studio Code digunakan sebagai code editor dalam proses penulisan source code website.
#### XAMPP dan MAMP
XAMPP dan MAMP digunakan sebagai web server lokal untuk menjalankan aplikasi Laravel dan database MySQL selama proses pengembangan website.
